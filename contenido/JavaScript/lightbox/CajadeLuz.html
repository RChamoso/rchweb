Lightbox JS v2.05

Ejemplo

Instrucciones de uso:

Parte 1 - Configuración

  1. Lightbox v2.0 utiliza el marco de prototipos y la biblioteca de efectos de Scriptaculous. Usted tendrá que incluir estos tres archivos Javascript en su cabecera. 
    
     <script type="text/javascript" src="js/prototype.js"></script>
     <script type="text/javascript" src="js/scriptaculous.js?load=effects,builder"></script>
     <script type="text/javascript" src="js/lightbox.js"></script>
	 
       Prototype.js es un Framework de JavaScript que tiene como objetivo
       facilitar el desarrollo de aplicaciones web dinámicas.
  2. Incluya el archivo de la mesa de luz CSS (o anexar su hoja de estilo activo con los estilos de la mesa de luz). 
    
     <link rel="stylesheet" href="css/lightbox.css" type="text/css" media="screen" />
  3. Compruebe la CSS y asegúrese de que la referencia prevlabel.gif and nextlabel.gif archivos se encuentran en el lugar correcto. Además, asegúrese de que el loading.gif y closelabel.gif archivos como se indica en la parte superior de la lightbox.js archivo se encuentran en el lugar correcto.

Parte 2 - Activar

  1. Añadir un rel="lightbox" atributo de cualquier etiqueta de enlace para activar la caja de luz. Por ejemplo: 
    
      <a href="images/image-1.jpg" rel="lightbox" title="my caption">image #1</a>
    Opcional: Utilice el title atributo si desea mostrar un título.
  2. Si usted tiene un conjunto de imágenes relacionadas entre sí que le gustaría grupo, siga el paso uno, sino, además, de un nombre de grupo entre corchetes en el atributo rel. Por ejemplo: 
    
     <a href="images/image-1.jpg" rel="lightbox[roadtrip]">image #1</a>
     <a href="images/image-2.jpg" rel="lightbox[roadtrip]">image #2</a>
     <a href="images/image-3.jpg" rel="lightbox[roadtrip]">image #3</a>
    No hay límites al número de conjuntos de imágenes por página o número de imágenes que se permiten en cada conjunto.
