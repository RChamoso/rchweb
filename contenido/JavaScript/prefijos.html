Cómo detectar prefijos propietarios con JavaScript

CSS, javascript, WebKit, prefijo, propietario, js, moz, element, vendor, prefix

Más allá de la opinión que tengamos de los prefijos propietarios, tenemos que vivir con ellos y ocasionalmente usarlo para que las cosas funcionen. Estos prefijos pueden ser usados de dos maneras: el formato CSS (por ejemplo "-moz-", como en -moz-element) y el formato JS ("navigator.mozApps"). Pero aquí te mostraremos una manera inteligente de detectar esos prefijos a traves de JavaScript.

Ver demostración >>

El primer paso es obtener el elemento HTML CSSStyleDeclaration:

PLAIN TEXT
  1. var styles = window.getComputedStyle(document.documentElement, ''),

El siguiente paso es convertirlo en un objeto array y buscar por un tipo de prefijo que sepamos:

PLAIN TEXT
  1. pre = (Array.prototype.slice
  2.       .call(styles)
  3.       .join('')
  4.       .match(/-(moz|webkit|ms)-/) || (styles.OLink === '' && ['', 'o'])
  5.     )[1]

Con el prefijo CSS encontrado, tomar el prefijo con el formato JS es simple:

PLAIN TEXT
  1. return {
  2.     dom: dom,
  3.     lowercase: pre,
  4.     css: '-' + pre + '-',
  5.     js: pre[0].toUpperCase() + pre.substr(1)
  6.   }

El objeto devuelto nos provee de un objeto que se ve de la siguiente manera:

PLAIN TEXT
  1. Object {dom: "WebKit", lowercase: "webkit", css: "-webkit-", js: "Webkit"}

Una completa representación del prefijado propietario para el navegador anfitrión. Aquí está el código restante:

PLAIN TEXT
  1. var prefix = (function () {
  2.   var styles = window.getComputedStyle(document.documentElement, ''),
  3.     pre = (Array.prototype.slice
  4.       .call(styles)
  5.       .join('')
  6.       .match(/-(moz|webkit|ms)-/) || (styles.OLink === '' && ['', 'o'])
  7.     )[1],
  8.     dom = ('WebKit|Moz|MS|O').match(new RegExp('(' + pre + ')', 'i'))[1];
  9.   return {
  10.     dom: dom,
  11.     lowercase: pre,
  12.     css: '-' + pre + '-',
  13.     js: pre[0].toUpperCase() + pre.substr(1)
  14.   };
  15. })();

Y eso es todo ¡Hasta la próxima!
